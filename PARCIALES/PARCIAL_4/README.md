#  Parcial 4 – Gestion de Productos (PHP + MySQL)

Este proyecto corresponde al **Parcial 4** de la asignatura de **Desarrollo de Software VII**, desarrollado por **Antonio Reyna (Grupo 1LS231)**.

##  Descripcion
Sistema sencillo para la **gestion de productos** usando **PHP, MySQL y Laragon**.  
Permite registrar, listar, editar y eliminar productos desde una interfaz basica con conexión a base de datos.
